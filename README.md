# AI Apps Hub Dashboard

Professional Streamlit homepage dashboard for AI assets with:

- Tabular homepage view
- Clickable `Asset Name` that opens a dedicated detail page
- Data loaded from `Asset Detail.xlsx` with:
  - header at row 2
  - data starting row 4
- Auto-appended dummy rows if the source has fewer entries

## Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
